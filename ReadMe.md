# 青鸟项目-1903班成员(一张三李一陈一琚)

## 一、项目目录以及详解

① 文件夹命名规则:

​	web: 前端首页界面

​	admin: 后台系统管理界面

​	server: node服务器端接口

​	moblie:移动端界面

② 默认文件:

​	.git: git初始化

​	.gitgnore: 忽略上传文件

​	ReadMe.md 本文件项目详情介绍

③ 项目命名规范:

​	一切以小驼峰命名规则 注释:  adminUser(管理员用户) userName(用户名称) passWord(密码)

​	文件名 .vue 文件以 vue官方为例: 首字母大写大驼峰命名规则文件

④ 所用的的语言框架

前端UI构建: Vue (框架) + Element UI (pc端) + Ionic (移动端) 

服务器端: Node(语言) + express(框架)
